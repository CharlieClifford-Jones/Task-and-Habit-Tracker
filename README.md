Task and Habit Tracker

Overview:

This project is a Task and Habit Tracker built as a Single Page Application (SPA) using HTML, CSS and JavaScript.

The aim of the application is to help users stay organised by allowing them to add tasks and habits, and track whether they have been completed or not.

Features:
- Add tasks and habits  
- Move tasks (complete / incomplete)  
- Move habits (kept up / broken)  
- Delete items  
- Save data using localStorage  
- Export and import data using JSON  

How to Use:
- Type a task or habit into the input box  
- Click "Add" to create it  
- Use the buttons to move items between sections  
- Use the save/load section to export or import data  

Technologies Used:
- HTML  
- CSS  
- JavaScript  

Notes:
The application runs in the browser and does not require an account or internet connection after loading.
